AEM Package Maven Plugin

aem-package:upload

Full name:

com.icfolson.maven.plugins:aem-package-maven-plugin:0.6.0:upload

Description:

(no description)

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: install.

Optional Parameters

Name Type Since Description
connectTimeout Integer - Connection timeout in milliseconds.
Default value is: 30000.
contextPath String - AEM context path.
fileName String - AEM package name.
Default value is: ${project.build.directory}/${project.build.finalName}.zip.
User property is: aem.package.fileName.
force boolean - Force upload of AEM package even if it already exists.
Default value is: true.
User property is: aem.package.force.
host String - AEM host name.
Default value is: localhost.
password String - AEM password.
Default value is: admin.
port Integer - AEM port number.
Default value is: 4502.
quiet boolean - Quiet logging when executing package command.
Default value is: false.
User property is: aem.package.quiet.
readTimeout Integer - Read timeout in milliseconds.
Default value is: 30000.
retryDelay Integer - Delay in milliseconds before retrying package command.
Default value is: 1000.
retryLimit Integer - Number of times to retry package command before aborting.
Default value is: 5.
secure boolean - Connect to Package Manager using https scheme.
Default value is: false.
skip Boolean - Skip execution of the plugin.
Default value is: false.
User property is: aem.package.skip.
username String - AEM user name.
Default value is: admin.
